# 🏦 Bank Management System (CLI) - Python

A Smart Command Line Interface (CLI) based Bank Management System developed using Python.
This project demonstrates the use of Object-Oriented Programming (OOP), File Handling, Exception Handling, and Git version control.

## 📌 Project Objective
To build a real-world CLI application that can:

- Create bank accounts
- Deposit money
- Withdraw money
- Check account balance
- Store and retrieve data using JSON file

## 🚀 Features
✔ Create New Account  
✔ Deposit Money  
✔ Withdraw Money  
✔ Check Balance  
✔ Data Persistence using JSON  
✔ Error Handling for Invalid Inputs  
✔ Menu-Driven Interface  

## 🛠 Technologies Used
- Python 3
- Object-Oriented Programming (Class & Methods)
- File Handling (JSON)
- Exception Handling
- Git & GitHub
